OpenVPN - Debian -> Nutzer anlegen/.ovpn Zertifikat erstellen

  • 1. OpenVPN Konfig-Menü öffnen

    CLI Befehl: bash openvpn-install.sh

  • 2. SSH Konfig-Datei zum bearbeiten öffnen

    CLI Befehl: nano /etc/ssh/sshd_config

  • 3. sshd_config bearbeiten

    Kommentar entfernen: #PermitRootLogin yes ➨ PermitRootLogin yes

    Aus Sicherheitsgründen sollte der Kommentar nach dem Download der .ovpn Datei wieder hinzugefügt werden
  • 4. SSH Dienst neustarten

    CLI Befehl: /etc/init.d/ssh restart
